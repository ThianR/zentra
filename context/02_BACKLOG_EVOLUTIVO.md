# Backlog General del Proyecto Zentra

- Implementar flujo base: Generar XML -> Firmar -> Enviar -> Consultar -> KuDE.

## Plan C: Enterprise & White-label
- Soporte HSM (Hardware Security Module).
- Personalizacion de marca (White-label).
- Despliegues dedicados (Single-tenant).
